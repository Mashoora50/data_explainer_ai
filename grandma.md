I developed a web-based application that allows users to interact with structured data files such as Excel spreadsheets,using natural language questions. The aim of the web is to make data analysis more accessible by translating complex datasets into simple human readable insights.

Instead of manually analyzing rows and columns, users can ask questions like 
What trends are present in the data? 
Which category performs the best?
what are the column names what has been stored and what is data type of column?
The system processes the data and responds with clear,easy tounderstand explanations.
there is multi stage AI pipeline with distinct analytical and generative components:
1) the analyst:The system processes the uploaded dataset, looks for patterns, aggregates values, and extracts useful insights.
2)Explainer:The system takes  insights and converts them into a natural language explanation the user can understand.
Technologies Used

To implement this system I used the following technologies:

Python – Core programming language for backend logic
Flask – Web framework used to build and serve the application
Groq AI – Provides fast language model responses for generating explanations
ChromaDB – Used as a vector database to efficiently store and retrieve relevant information
LangChain – Helps orchestrate and connect the different AI components into a unified workflow

I also implemented basic safety constraints to ensure the system only responds within the context of the uploaded dataset. This prevents irrelevant or misleading queries and ensures the outputs remain focused and reliable.

how to use:

The user experience is designed to be very simple:

Upload a dataset (e.g., Excel or CSV file) and load
Enter a natural language question about the data
Receive a clear human friendly explanation generated by the system
